## 二、间距系统

### 2.1 使用频率统计

| 值 | 使用次数 | 占比 | 推荐场景 |
|----|---------|------|---------|
| **20rpx** | 1899 | 34% | 常用间距、列表项间距 |
| **32rpx** | 1123 | 20% | 页面边距、卡片内边距 |
| 24rpx | 678 | 12% | 中等间距 |
| 16rpx | 534 | 10% | 小间距、图标与文字间距 |
| 8rpx | 312 | 6% | 紧凑间距 |
| 40rpx | 245 | 4% | 大间距、区块分隔 |

### 2.2 使用规范

```scss
// 页面容器
.page {
  padding: 32rpx;
  background: $bgc-page;
}

// 卡片
.card {
  padding: 32rpx;
  margin-bottom: 20rpx;
  background: $bgc-white;
  border-radius: 8rpx;
}

// 列表项
.list-item {
  padding: 20rpx 32rpx;

  + .list-item {
    border-top: 1rpx solid $border-grey-light;
  }
}
```
